发表于 2009-12-28 | 分类于 Linux |

perl自带的函数远不及php多，但功能确实很强大，实现起来也只有几行代码。

perl中的目录操作函数有：

1. mkdir 函数

调用语法：mkdir (dirname, permisions)
dirname：将要创建的目录名，可以为字符串或表达式。
permisions：八进制数，指定目录的访问权限。其值和意义如下：

权限值 权限
4000 运行时设置用户 ID
2000 运行时设置组 ID
1000 粘贴位
0400 拥有者读权限
0200 拥有者写权限
0100 拥有者执行权限
0040 组读权限
0020 组写权限
0010 组执行权限
0004 所有人读权限
0002 所有人写权限
0001 所有人执行权限
