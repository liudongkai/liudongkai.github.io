еҸӮж•°пјҡ

  • Reference жҳҜдҪңдёәеҒҸз§»йҮҸеҸӮз…§зі»зҡ„еј•з”ЁеҢәеҹҹпјҢе®ғеҝ…йЎ»жҳҜеҚ•е…ғж јжҲ–зӣёиҝһеҚ•е…ғж јеҢәеҹҹзҡ„еј•з”Ёпјӣ
  • Rows жҳҜзӣёеҜ№дәҺеҒҸз§»йҮҸеҸӮз…§зі»зҡ„е·ҰдёҠи§’еҚ•е…ғж јпјҢдёҠ(дёӢ)еҒҸз§»зҡ„иЎҢж•°гҖӮеҰӮжһңдҪҝз”Ё5 дҪңдёәеҸӮж•°RowsпјҢеҲҷиҜҙжҳҺзӣ®ж Үеј•з”ЁеҢәеҹҹзҡ„е·ҰдёҠи§’еҚ•е…ғж јжҜ”reference дҪҺ5 иЎҢгҖӮиЎҢж•°еҸҜдёәжӯЈж•°(д»ЈиЎЁеңЁиө·е§Ӣеј•з”Ёзҡ„дёӢж–№)жҲ–иҙҹж•°(д»ЈиЎЁеңЁиө·е§Ӣеј•з”Ёзҡ„дёҠж–№)пјӣ
  • Cols жҳҜзӣёеҜ№дәҺеҒҸз§»йҮҸеҸӮз…§зі»зҡ„е·ҰдёҠи§’еҚ•е…ғж јпјҢе·Ұ(еҸі)еҒҸз§»зҡ„еҲ—ж•°гҖӮеҰӮжһңдҪҝз”Ё5 дҪңдёәеҸӮж•°ColsпјҢеҲҷиҜҙжҳҺзӣ®ж Үеј•з”ЁеҢәеҹҹзҡ„е·ҰдёҠи§’зҡ„еҚ•е…ғж јжҜ”reference йқ еҸі5 еҲ—гҖӮеҲ—ж•°еҸҜдёәжӯЈж•°(д»ЈиЎЁеңЁиө·е§Ӣеј•з”Ёзҡ„еҸіиҫ№)жҲ–иҙҹж•°(д»ЈиЎЁеңЁиө·е§Ӣеј•з”Ёзҡ„е·Ұиҫ№)пјӣ
  • Height жҳҜиҰҒиҝ”еӣһзҡ„еј•з”ЁеҢәеҹҹзҡ„иЎҢж•°пјҢHeight еҝ…йЎ»дёәжӯЈж•°пјӣ
  • Width жҳҜиҰҒиҝ”еӣһзҡ„еј•з”ЁеҢәеҹҹзҡ„еҲ—ж•°пјҢWidth еҝ…йЎ»дёәжӯЈж•°гҖӮ
